# Sun ChatBot – Your Personal Task Manager

> “Never leave that till tomorrow which you can do today.” – Benjamin Franklin ([source](https://www.brainyquote.com/quotes/benjamin_franklin_151632))

Sun helps you **free your mind** by remembering your tasks so you don’t have to. It’s:

- **Text-based**
- *Easy to learn*
- ~~Slow~~ **SUPER FAST** to use

Getting started is simple:

1. Download it from [here](https://github.com/SUN-HEQIAO/ip/releases/tag/A-Release)
2. Double-click it
3. Add your tasks
4. Let it remember your tasks for you 😉

And yes – it’s **completely FREE**!


## Features

* [x] Add tasks: Create todos, deadlines, or events quickly.
* [x] Mark/Unmark tasks: Designate tasks as completed or pending tasks.
* [x] Task overview: View all your tasks in an organized list.
* [x] Delete tasks: Remove tasks when they are no longer relevant.
* [x] Undo actions: Undo mistakenly managed task.
* [x] Find tasks: Search for specific tasks by keywords.
* [x] Help command: View all executable commands anytime.
